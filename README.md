# Netflix-Clone

An application based on the general idea and design of Netflix. I have added some custom features like loading state when call APIs, skeleton views, alphabetical sorting, hiding/showing toggle for adult contents when searching, and showing success/error/info alerts (including exception handling), etc.

## Technologies and resources in use
1. Codebase: Swift, UIKit, Interface Builder, Core Data
2. Architecture: Clean VIP (optional router)
3. 3rd party libaries: SVProgressHUD, SDWebImage, SkeletonView
4. APIs: https://www.themoviedb.org/
